# 模块一：AI 认知地基

<Badge variant="outline">3 周</Badge> <Badge variant="secondary" className="ml-2">约 10 小时</Badge>

## 这个模块要做什么

<KeyPoint emoji="🎯">
  在 3 周内，你将从「用过 AI 工具」跨越到「理解 AI 怎么工作并自己搭建一个 AI 应用」。
</KeyPoint>

**项目实战**：做一个可定制人设的 ChatGPT Web 应用。用户可以自定义 AI 的「人设」（比如「你是一个儿童教育专家」），调节温度和最大 Token 数，保存和加载对话历史。最后部署上线，生成一个可分享的链接。

**核心认知**：AI 不是魔法。它不「知道」任何事，它只是在「猜下一个字」。理解了这一点，你就超过了 90% 的 AI 用户。

## 你需要准备什么

| 工具 | 用途 | 获取方式 |
|------|------|---------|
| Python 3.9+ | 运行环境 | 你应该已经有了 |
| Claude API Key 或 OpenAI API Key | 调用 AI 模型 | console.anthropic.com 或 platform.openai.com |
| Streamlit | 把 Python 脚本变成 Web 应用 | `pip install streamlit` |
| 一个代码编辑器 | 写代码 | 推荐 Cursor 或 VS Code |
